## reverse(str)

Reverse the order of the characters of the string. Handles surrogate pairs!


### Arguments
* `str` *(string)*: The string to reverse

### Returns
*(string)*: Returns a reversed string 


### Example
```js
replace('my string')
// => 'gnirts ym'

replace('ma単ana ma単ana')
// => 'ana単am ana単am'
```
