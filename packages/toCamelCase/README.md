## toCamelCase(str)

Apply camel casing to the string


### Arguments
* `str` *(string)*: string to apply camel case

### Returns
*(string)*: Returns a camel cased string 


### Example
```js
toCamelCase('my string')
// => 'myString'

toCamelCase('my_string')
// => 'myString'

toCamelCase('my-string')
// => 'myString'
```
