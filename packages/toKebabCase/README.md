## toKebabCase(str)

Apply kebab casing to the string


### Arguments
* `str` *(string)*: string to apply kebab case

### Returns
*(string)*: Returns a kebab cased string 


### Example
```js
toKebabCase('my string')
// => 'my-string'

toKebabCase('my_string')
// => 'my-string'

toKebabCase('myString')
// => 'my-string'
```
