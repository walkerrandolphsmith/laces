## collapse(str)

Remove all whitespace from the string.


### Arguments
* `str` *(string)*: The string to remove whitespace from

### Returns
*(string)*: Returns a string absent of whitespace


### Example
```js
collapse('my string')
// => 'mystring'

collapse('my string\n another line')
// => 'mystringanotherline'
```
