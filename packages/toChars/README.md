## toChars(str)

Create an array of characters from the string


### Arguments
* `str` *(string)*: string to split

### Returns
*(Array<string>)*: Returns an array of characters 


### Example
```js
toChars('my string')
// => ['m', 'y', ' ', 's', 't', 'r', 'i', 'n', 'g']
```
