## toSpaceCase(str)

Apply space casing to the string


### Arguments
* `str` *(string)*: string to apply space case

### Returns
*(string)*: Returns a space cased string 


### Example
```js
toSpaceCase('my_string')
// => 'my string'

toSpaceCase('my-string')
// => 'my string'

toSpaceCase('myString')
// => 'my string'
```
