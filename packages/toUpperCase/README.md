## toUpperCase(str)

Convert each character in the string to upper case


### Arguments
* `str` *(string)*: string to apply upper case

### Returns
*(string)*: Returns a upper cased string 


### Example
```js
toUpperCase('MY STRING')
// => 'MY STRING'

toUpperCase('my StrinG')
// => 'MY STRING'
```
