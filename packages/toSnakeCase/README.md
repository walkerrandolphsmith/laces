## toSnakeCase(str)

Apply snake casing to the string


### Arguments
* `str` *(string)*: string to apply snake case

### Returns
*(string)*: Returns a snake cased string 


### Example
```js
toSnakeCase('my string')
// => 'my_string'

toSnakeCase('my-string')
// => 'my_string'

toSnakeCase('myString')
// => 'my_string'
```
